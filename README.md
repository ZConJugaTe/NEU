# NEU Maze Game

欢迎来到 **NEU Maze Game** 项目！这是一个使用 JavaScript 和 HTML 实现的迷宫游戏，玩家可以通过键盘控制角色移动，目标是找到从起点到终点的路径。

## 项目链接

[在线演示](https://zconjugate.github.io/NEU/)

## 功能

- 自动寻路功能：使用 A* 算法找到最短路径。
- 迷宫动态生成：每次游戏开始时生成新的迷宫。
- 键盘控制：使用方向键控制角色移动。
- 视觉效果：简洁的用户界面和迷宫渲染。

## 技术栈

- HTML
- CSS
- JavaScript
